# Student-Social-Media-Behavior-Impact-Analysis-Dashboard-Power-BI-
Through this Project i Developed an interactive Power BI dashboard to analyze the impact of social media usage on students’ mental health, academic performance, and lifestyle patterns.

In this project i 200+ analyze Students Social Media Addiction and their impact on students daily behaviour. In this project i analyze two tables one is of studets details and other is of plateform details.

Created calculated columns and DAX measures (Avg Usage, Addiction Count, % Academic Impact) to derive actionable insights.

Implemented time intelligence using a custom Date Table to simulate usage trends and enable temporal analysis.

Built multiple dashboards (Executive Overview, Mental Health, Academic Impact, Relationships) with visuals such as scatter plots, matrices, and drill-through reports.

Enabled user interactivity using slicers, bookmarks, and drill-through to provide detailed student-level insights.

Identified key correlations between high social media usage, lower sleep hours, increased addiction, and declining mental health.

Identified that high usage users (>5 hrs/day) show 20–30% higher addiction scores and lower mental health ratings, highlighting a negative correlation between usage and well-being
